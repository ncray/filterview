What is FilterView?

FilterView is a programming model and associated implementation for interactive visualization of large, distributed datasets. Users specify a filter function that processes a table to extract an intermediate set of rows satisfying some condition, and a view function that updates a templated <div> element using these rows. An extremely large class of data visualization problems are expressible in this model.

Programs written in this functional style may easily be parallelized and executed on large clusters of commodity machines, with incremental requests for individual rows handled by remote JSON requests. All visualization is performed in the browser, enabling structured rendering of rich elements. UI elements are auto-generated by introspecting type information, and UI states correspond to versions of the filter function. The consequence is that programmers with minimal experience in Javascript idiosyncrasies or UI design may easily generate interactive, collaborative visualizations. We present our implementation of FilterView which runs on any POSIX computer and some examples of visualization of both local and distributed datasets.
